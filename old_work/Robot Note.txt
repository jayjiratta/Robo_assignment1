Robot Note

AP Mode (Access Point Mode):
ep_robot.initialize(conn_type="ap")
โหมดนี้ทำให้หุ่นยนต์ทำหน้าที่เป็น Access Point (AP) หรือสร้างเครือข่าย Wi-Fi ของตัวเอง
อุปกรณ์ที่ควบคุม (เช่น สมาร์ทโฟนหรือคอมพิวเตอร์) ต้องเชื่อมต่อกับเครือข่าย Wi-Fi ที่หุ่นยนต์สร้างขึ้น

STA Mode (Station Mode):
ep_robot.initialize(conn_type="sta")
โหมดนี้ทำให้หุ่นยนต์ทำหน้าที่เป็นไคลเอนต์หรือเชื่อมต่อกับเครือข่าย Wi-Fi ภายนอกที่มีอยู่แล้ว (เช่น เชื่อมต่อกับเราเตอร์ที่บ้านหรือที่ทำงาน)
หุ่นยนต์และอุปกรณ์ที่ควบคุมต้องเชื่อมต่อกับเครือข่าย Wi-Fi เดียวกัน
